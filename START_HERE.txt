⚡ БЫСТРЫЙ СТАРТ (5 МИНУТ):

1. Проверка системы:
   $ python check_system.py

2. Попробовать демонстрацию:
   $ python demo.py
   
3. Запустить систему:
   $ python main.py all
   
4. Открыть в браузере:
   - API:      http://localhost:8000
   - Dashboard: http://localhost:8501
